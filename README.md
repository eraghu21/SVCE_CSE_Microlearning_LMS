# Flask LMS

This is a Learning Management System (LMS) built using Flask.

## Features
- Admin / Teacher / Student login system
- Upload student Excel files
- Assign YouTube learning videos
- Track student progress
- Auto-generate and download certificates after video completion

## To Run

```bash
pip install -r requirements.txt
python app.py
```

## Deployment

Deploy easily on [Render.com](https://render.com) or PythonAnywhere.
